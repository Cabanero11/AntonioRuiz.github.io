Django Estrenos cine

Aplicacion web desarollado mediante el Framework Django y el uso de Selenium asi como GithubActions. Donde se obtiene mediante 2 scripts con Selenium los datos de paginas de cine, para obtener los horarios y link de compra, para comparar los horarios de ambos cines.
El proyecto se encuentra en Github

En la siguiente imagen se observan todos los horarios para una pelicula en ambos cines

En este caso mediante Github Actions se automatiza el proceso de despliegue y actualizacion de los datos mediante GithubActions y Jekyll donde se instalan todas las herramientas necesarias en un entorno ubuntu, para ejecutar los diferentes scripts en Python

En esta parte se ven todos los procesos que realiza Github Actions para preparar el entorno y ejecutar los scritps.
