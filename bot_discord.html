Bot Discord Sam

Un bot de Discord, servicio de mensajería instantánea y chat de voz VolP, en cual es capaz realizar diferentes funciones de entretenimiento en servidores como reproducir musica, un sistema de economia,mandar memes , dar el tiempo empleando la API de Open Weather Map, asi como mandar fotos de gatitos super monos usando la API proporcionada The Cat Api.

Para realizar este Bot, se ha empleado Python, y varias liberias como: 'discord' para crear el bot, 'yt_dlp' para descargar y reproducir musica, 'asyncio' para realizar operaciones asincronas, 'json' para guardar en un .json las monedas de los usuarios.
